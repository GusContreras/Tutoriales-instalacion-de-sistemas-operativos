Sistemas a utilizar

Para esta práctica utilizaremos el sistema operativo Lubuntu y el sistema operativo Windows 10.
Lubuntu fue elegido por ser un sistema de bajos recursos ideal para computadores con bajas características.
Por otra parte, Windows 10 es un sistema operativo moderno altamente gráfico, por tal motivo requiere de una máquina más potente.

 
 

Creación de USB Booteable

Para la realización de esta práctica se recomienda utilizar una memoria USB de 8 Gb o más, o en su caso disponer de un CD. Lo primero que tenemos que hacer es descargar un software que nos sirva como herramienta para crear USB booteables, en está práctica utilizaremos el software Rufus el cual es bastante fácil de usar.

Utilizar Rufus es más fácil de lo que parece puesto que sólo tendremos que elegir la imagen ISO y Rufus rellenará el resto de los campos de manera automática, una vez llenados los campos solo basta con darle al botón de "empezar" y esperar a que la USB esté lista.

 
 

Acceder a la BIOS

Lo primero que tenemos que hacer es iniciar desde la BIOS en el computador que queremos instalar el sistema operativo. Para acceder a la BIOS se requiere presionar una tecla al momento de arranque del computador, ésta varía según el fabricante, siendo f2 o f12 las más populares.

Una vez que accedimos a la BIOS, tenemos que elegir el apartado de "Boot Device Priority" y colocar nuestra USB como la prioridad más Alta como se muestra en la imagen. Ya que nuestra USB tenga la mayor prioridad, guardamos los cambios y salimos.

 
 

Instalar Lubuntu

Una vez que el computador arranque con nuestra USB, se iniciará el instalador gráfico, en primer lugar nos pregunta el lenguaje.

Ya que escogimos un lenguaje nos aparecerán 5 opciones de las cuales vamos a elegir la opción de "Instalar Lubuntu".

Posteriormente se nos preguntará sobre el tipo de instalación. Si nuestro PC tiene almacenamiento limitado se recomienda seleccionar la "instalación mínima", de lo contrario podemos elegir la "instalación normal".

Si tenemos ya instalado otro Sistema Operativo y deseamos solo mantener Lubuntu, seleccionamos la opción de "Borrar disco e instalar Lubuntu".

En seguida pulsaremos en el mapa que se despliega nuestra ubicación.

Se nos pedirá crear un usuario llenando un pequeño formulario.

Ahora solo queda esperar a que se instale el sistema Opertivo.

Una vez instalado el sistema, retiramos la USB y esperamos a que se reinicie.

Con eso damos por concluida la instalación de Lubuntu.

 
 

Instalar Windows 10

Una vez que arranque nuestro PC con la memoria Booteable, lo primero que tendremos que hacer es determinar el idioma, el formato de hora y el teclado.

Posterirormente se nos solicitará una clave de producto, si compraste Windows desde la página oficial la clave se te ha enviado al correo, si compraste un disco la clave viene en él, si no dispones aún de una clave puedes elegir la opción de "No tengo una clave de producto" o usar una clave genérica con servicios limitados.

Hayamos colocado la clave o no, enseguida se nos mostrarán los términos y condiciones, los cuales tendremos que aceptar para seguir con la instalación.

Una vez que se acepten los términos y condiciones se nos preguntará el tipo de instalación. Existen 2 opciones, la primera nos permite migrar nuestros archivos y aplicaciones, esta opción solo funciona si ya tenemos instalada una versión compatible de Windows, por ejemplo Windows 7. La segunda opción no migrará nada, ideal por si quieres empezar desde cero o vas a vender el equipo.

Despúes de determinar el tipo de instalación tendremos que elegir la partición donde queremos instalar el Sistema Operativo.

Nos dará una advertencia sobre el movimiento de archivos en caso de que dispongamos de una versión antigua de Windows.

Aquí es cuando comienza a instalarse Windows, recomiendo ser paciente ya que la instalación puede durar varias horas.

Después de haber terminado la instalación, se nos volverá a preguntar la región.

Ahora tendremos que elegir una distribución de teclado.

Enseguida tienes que determinar si la PC será para uso personal o forma parte de una red en una organización.

Por último, determinaremos nuestra configuración de privacidad, se recomienda leer este apartado y solo activar las que creas necesarias.

Una vez determinada nuestra configuración de privacidad, tendremos que esperar unos minutos, que en algunos casos se pueden extender a algunas horas.

Despúes de la espera nuestro Windows 10 habrá sido instalado.
