Introducción

Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código. Se trata de un lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma.
Para descargar python has click aquí
Django es un framework web diseñado para realizar aplicaciones de manera fácil y rápida. Está escrito en Python y tiene una comunidad muy amplia, que está en continuo crecimiento.
Para conseguir Django necesitamos unos comandos los cuales podemos encontrar aquí El programa realizado es sencillo ya que solo será una variante del típico "Hola Mundo".

 
 

Instalación en Windows 10

Lo primero que tenemos que hacer es instalar Python, para ello abrimos el link que se dejo arriba en la introducción y descargamos y ejecutamos la versión de Python que deesemos en nuestro equipo.

Iniciamos la instalación y esperamos unos minutos a que la instalación termine.

Después de unos minutos tendremos Python instalado.

Ahora crearemos un entorno virtual con el comando "-m venv nombre_del_entorno"

Con el comando "nombre_del_entorno\Scripts\activate" activaremos el entorno para posteriormente utilizar el comando de isntalación de Django el cual aparece en la pagína marcarda en la introducción, en este caso fue "pip install Django==3.0.3"

Una vez instalado Django creamos un proyecto con el comando "django-admin startproject nombre_del_proyecto"

Nos movemos a la carpeta de nuestro proyecto y corremos el servidor con el comando "python manage.py runserver"

Abrimos el navegador y nos vamos a la dirección que marca la consola.

Por último agregamos una nueva vista usando el código antes mostrado.

 
 

Instalar en Lubuntu

Lo primero que tenemos que hacer es instalar Python, para ello utilizaremos el comando "sudo apt install python3"

Ahora instalaremos el pip de python con el comando "sudo apt install python3-pip"

Ahora con el comando "pip install virtualenv" instalaremos una herramienta para poder crear entornos virtuales.

Con el comando "-m virtualenv Nombre_del_entorno" crearemos nuestro entorno virtual.

Ahora con el comando "source Nombre_del_entorno/bin/activate" activaremos nuestro entorno virtual.

Después de activar nuestro entorno podremos instalar Django con el comando "pip3 install django"

Esperamos unos minutos a que la instalación términe, para posteriormente crear un proyecto con el comando "django-admin startproject nombre_del_proyecto".

Nos movemos a la carpeta del proyecto y utilizamos el comando "python manage.py runserver" para correr nuestro servidor.

Abrimos nuestro navegador con la dirección que nos muestra la consola y con eso terminamos la instalación de Django.
