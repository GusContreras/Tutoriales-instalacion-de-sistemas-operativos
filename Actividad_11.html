Introducción

Ubuntu Server es una variante de Ubuntu y está dedicada especialmente para su uso en servidores. El uso de Ubuntu como servidor se ha extendido mucho en los últimos años, sobretodo para el uso de servidores web.
Para descargar Ubuntu Server has click aquí

 
 

Instalación de Ubuntu Server

Lo primero que tenemos que hacer es elegir el idioma de preferencia.

Elegimos la distribución de teclado.

Dejamos por defecto los adaptadores de red.

Damos click en hecho, ya que en mi caso no requiero de un proxy para hacer la conexión.

Dejamos el espejo por defecto y damos click en hecho.

Ahora se nos pregunta donde se instalará el sistema operativo, en este caso elegimos el disco entero y damos click en hecho.

Se desplegará un mensaje de advertencia, debido a que todo el contenido del disco será borrado, simplemente le damos en continuar.

Creamos nuestro usuario principal rellenando el formulario.

Damos click en hecho.

Como no requerimos ningún complemento solo damos click en hecho.

Esperamos a que se realice la instalación.

Una vez que la instalación haya finalizado damos click en reiniciar.

Una vez se haya reiniciado podremos iniciar sesión con el usuario que anteriormente creamos.

Iniciamos sesión tranquilamente.

 
 

Creación de Usuarios y Grupos

Lo primero que tenemos que hacer es utilizar el comando "sudo -s" para iniciar como administrador.

Creamos 4 grupos usando el comando "groupadd -g ID NombreDelGrupo".

Creamos 10 usuarios con el comando "useradd -u IdUsuario -G IdGrupo NombreDeUsuario".
